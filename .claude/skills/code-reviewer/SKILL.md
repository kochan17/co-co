# Code Reviewer Skill

## Description
コードの品質を向上させるためのレビュースキル。セキュリティ、パフォーマンス、可読性の観点からコードを分析します。

## When to Use
- プルリクエストのレビュー時
- コード品質の改善が必要な場合
- セキュリティ脆弱性のチェック

## Tools Available
- Read: コードファイルの読み取り
- Grep: パターンマッチング検索
- Bash: 静的解析ツールの実行

## Instructions
以下の観点でコードをレビューしてください：

1. **セキュリティ**
   - SQLインジェクション、XSS等の脆弱性
   - 認証・認可の適切な実装
   - 機密情報の漏洩リスク

2. **パフォーマンス**
   - O(n²)以上のアルゴリズム
   - 不要なAPI呼び出し
   - メモリリークの可能性

3. **可読性・保守性**
   - 命名規則の一貫性
   - 関数の単一責任原則
   - コメントの適切性

4. **ベストプラクティス**
   - フレームワーク固有の推奨事項
   - エラーハンドリング
   - テストカバレッジ

## Output Format
```markdown
## Code Review Report

### ✅ Good Practices
- [良い点を列挙]

### ⚠️ Issues Found
- [問題点と修正提案]

### 🔒 Security Concerns
- [セキュリティ関連の懸念点]

### 🚀 Performance Suggestions
- [パフォーマンス改善提案]
```
