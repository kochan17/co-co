# 研修プラン 3つの定義パターン

各企業のニーズやリテラシーに合わせて使い分けられるよう、3つの切り口でプランを構造化しました。

---

## 1. 【期間パターン】
「いつまでに、どの程度の時間で」を重視する企業向け。

| 期間 | フェーズ | 主な内容 |
| :--- | :--- | :--- |
| **1ヶ月（短期集中）** | 基礎・ツール構築 | 生成AIの原則習得に加え、**「自分専用AIツールの構築（GPTs, Gems, Copilot Agent等）」**まで実施。短期間でも実務に役立つ資産を残す。 |
| **2ヶ月（個人開発）** | 実践・自動化 | 業務の棚卸、GASやVBAを用いた個人の定型業務の自動化。 |
| **3ヶ月〜（組織浸透）** | チーム開発・内製化 | チームでのAIツール共同開発、RAG構築、組織的なAI運用体制の構築。 |

---

## 2. 【到達パターン】
「受講者がどう変わるか」という教育効果を重視する企業向け。

- **Lv.1：AIユーザー（基礎）**
  - AIを検索や文章作成のパートナーとして使いこなし、個人の生産性を20〜30%向上させる。
- **Lv.2：DXエグゼクティブ（応用）**
  - 自らノーコードツールやスクリプトを書き、自身の複雑な業務フローを自動化できる。
- **Lv.3：AIアーキテクト（プロフェッショナル）**
  - チームや部門全体の課題をAIで解決するシステム（エージェントや共有アプリ）を設計・構築できる。

---

## 3. 【支援形態パターン】
「自習」か「伴走」かというサポートの厚みを重視する企業向け。

- **スタンダードプラン（自走型）**
  - 動画教材 ＋ 質問対応（チャット）。リテラシーが高く、自ら進められる組織向け。
- **伴走サポートプラン（個別支援型）**
  - 週1回の1on1メンタリング ＋ 進捗管理。脱落を防ぎ、確実にアウトプットを出したい組織向け。
- **プロジェクトコーチング（チーム支援型）**
  - チーム単位でのグループコーチング ＋ 開発アドバイザリー。実際のプロダクト開発を成功させたい組織向け。

---

## 4. 【全プラン共通】定義すべき重要項目
どのパターンを採用する場合でも、以下の要素をセットにすることで「div/antigravityらしさ」と「安心感」を担保します。

- **「やりきる」伴走サポート体制**
  - 専任メンターによる進捗管理、回数無制限のチャット質問対応。
- **実務直結のケーススタディ（Before/After）**
  - 単なる操作説明ではなく、「どの業務が何時間減るか」というROI（投資対効果）の提示。
- **助成金活用サポート**
  - 実質負担を1/4（75%還付）程度まで抑えるための申請支援。
- **成果発表会（クロージング）**
  - 研修の最後にアウトプットを社内共有し、組織全体への波及効果を生む。
  - **※成果発表用の高品質なPPTXテンプレートは弊社にて提供。**

---

## 5. 【全プラン共通】定常サポート・カリキュラム

### 5.1 サービス内容（AI顧問）
- **週1回・30分間の定期MTG**を実施。
- 受講生の進捗に限らず、今後の活用方針やツール導入などの相談に柔軟に対応します。
- **※運用ルール**: 円滑な進行のため、質問事項は事前にチャット等でまとめていただく形式を推奨しております（未整理の場合も、お時間の許す限りその場で回答いたします）。

### 5.2 学習カリキュラムの核
どんなに期間が短いプランでも、以下の内容は必ずカリキュラムに含めます。

- **1. 生成AIの本質理解**
  - **定義**: 生成AIと従来のAIとの決定的な違い。
  - **仕組み**: どのようなロジックで回答が生成されるのか。
  - **リスク管理**: セキュリティ、機密情報保持、法的リスクへの向き合い方。

- **2. ツールの4分類とマッピング**
  | 分類 | 代表的なツール | 特徴・役割 |
  | :--- | :--- | :--- |
  | **チャット型 (Chat)** | ChatGPT, Gemini, Claude, Copilot | **「会話」**形式で日常業務（文章作成・調査）を汎用的に支援。 |
  | **ワークフロー型 (Workflow)** | Dify, n8n | 特定の手順を定義し、一連の処理を**「自動化」**する仕組み。 |
  | **エージェント型 (Agent)** | Claude Code, GitHub Copilot, Google AI Studio | 指示を与えると、計画〜実装まで**「自律的」**に遂行。 |
  | **開発型 (Development)** | Cursor, Antigravity | アプリケーションやツールの**「開発・実装」**に特化。 |

- **3. 各ツールの目利きと使い分け**
  - 「自分の課題にはどの分類が最適か」を判断できるスキルの養成。

---

## 6. 【プライシング】
プレミアムな価値提供と「内製化」を軸にした強気の価格設定案です。（※1名あたりの税別価格）

| プラン区分 | 標準価格（1名あたり/定価） | 助成金活用後（1名あたり/実質） | 主な対象・内容 |
| :--- | :--- | :--- | :--- |
| **短期（〜1M）** | **50万円 〜** | **12.5万円 〜** | 基礎習得、GPTs等のツール構築 |
| **中期（〜2M）** | **80万円 〜** | **20.0万円 〜** | 個人の業務自動化（GAS / VBA） |
| **長期（3M〜）** | **120万円 〜** | **30.0万円 〜** | チーム開発、RAG、内製化支援 |

> [!IMPORTANT]
> **「成果」で価格を正当化する**
> - 1ヶ月50万円は一見高額ですが、助成金を活用すれば実質12.5万円です。
> - 「月間30時間削減（約7.5万円分）」×「永続的な効果」を提示することで、2ヶ月以内に投資回収が可能であることを強調します。

> [!NOTE]
> **付加価値による調整**
> - 講師の専門性（エンジニア、DXコンサル）
> - 個別カスタマイズの有無（特定部門向け等）
