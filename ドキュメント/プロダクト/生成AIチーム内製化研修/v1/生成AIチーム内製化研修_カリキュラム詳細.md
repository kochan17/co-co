# 生成AI研修 カリキュラム詳細（チーム開発：プランB）

## 1. 研修の狙い
「社内にAI開発機能を構築し、外部依存（SaaS/外注）からの脱却と企業の独自進化を実現する」

## 2. カリキュラム構成（3ステップ）

### STEP 1: 開発基盤の構築 (B-1: Standard)
AIを単なるチャットツールではなく、開発パートナーとして活用するための土台を作ります。
- **Git/GitHub基礎**: バージョン管理の概念、リポジトリ作成、コミット、プッシュの習得。
- **VS Code / Agentic AI環境**: VS Codeの基本操作、Claude Code / Antigravityの設定と実務への導入。
- **AIガバナンス研修**: セキュリティポリシーに基づくAI活用のルール作り。

### STEP 2: 実践・内製化 (B-2: Advanced)
実務課題をAIと共に解決し、独自のツールを構築する力を養います。
- **AIプログラミング実践**: PythonやVBAを活用した業務自動化スクリプトの作成。
- **チーム開発の作法**: プルリクエストベースの開発フロー、AIによるコードレビューの体験。
- **ワークフロー自動化**: Difyやn8n、PowerAutomateを活用した業務プロセスのAI化。

### STEP 3: 組織トランスフォーメーション (B-3: Enterprise)
AI活用を組織全体に広げ、継続的な改善サイクルを構築します。
- **AI顧問サービス**: 継続的な技術相談と最新トレンドの共有。
- **社内推進リーダー育成**: 部門横断的な情報共有とAI活用事例の横展開。
- **カスタムAIエージェント運用**: 特定業務に特化した独自のAIエージェントの構築と保守。

## 3. サポート体制
- **メンタリング**: 週1回のオンラインセッション（進捗確認・技術指導）。
- **チャットサポート**: Slack/Teamsによる随時のQ&A対応。
- **コードレビュー**: 技術メンターによる成果物の品質チェック。

## 4. 目標とする到達像
- 独自の業務効率化ツールを自社で開発・メンテナンスできる。
- AI活用におけるセキュリティリスクを適切に管理・運用できる。
- AIを前提とした柔軟な組織文化が醸成されている。
