# プロジェクト概要

クライアント企業のセキュリティガイドに基づき、本プロジェクトで使用を想定しているツール群は以下の通りです。

### IDE
- VSCode
- Cursor
- Antigravity

### AI Assistant
- Claude code（Claude導入済みの場合）

- codex（ChatGPT導入済みの場合）

### Chat AI
- ChatGPT
- Claude
- Gemini
- MS Copilot

### Workflow
- Dify
- n8n
- PowerAutomate

### Repository
- GitHub
