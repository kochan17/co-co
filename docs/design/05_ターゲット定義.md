# ターゲット定義：CoCo全体 + 各サービス別

## CoCo全体のターゲットポジショニング

### 会社レベルのICP (Ideal Customer Profile)
```
Primary Target: "AI導入意欲はあるが、適切な実行パートナーがいない中小〜中堅企業"

企業規模: 従業員 20-300名
売上規模: 5億円-100億円
業界: 製造業、サービス業、小売業、建設業
地域: 関東、中部、関西の地方都市
IT成熟度: 基本的なシステム導入済み、クラウド利用開始済み
```

### CoCo のブランドポジション
```
"中小企業のためのAI内製化パートナー"

競合との差別化:
- vs 大手ITコンサル: 中小企業に特化、手頃な価格
- vs AIツールベンダー: 伴走型サポート、確実な成果
- vs フリーランス: 組織的対応、継続性保証
```

## サービス別ターゲットセグメンテーション

### 1. AIチーム開発伴走支援

#### Primary Target
**中小企業DX推進責任者 + 経営者（決裁者）**

```
企業プロフィール:
- 従業員数: 50-200名
- 業界: 製造業、物流業、サービス業
- 年商: 10億-50億円
- 所在地: 地方都市（東京以外）
- DX段階: 基本的なデジタル化は完了、次のステップを模索

担当者プロフィール:
- 役職: 総務部長、システム管理者、DX推進室長
- 年齢: 35-50歳
- 背景: IT企業出身ではないが、会社のIT化を推進
- 課題: AI導入の必要性は理解、具体的方法が不明
```

#### 具体的な業種セグメント
```
1. 製造業（町工場・中堅製造）
   - 課題: 熟練工の技能継承、品質管理の属人化
   - ニーズ: 生産効率化、予防保全、品質向上

2. 物流・倉庫業
   - 課題: 人手不足、配送最適化、在庫管理
   - ニーズ: 自動化、ルート最適化、需要予測

3. 建設・不動産業
   - 課題: 現場管理、安全管理、コスト管理
   - ニーズ: 現場自動化、安全性向上、収益最大化
```

### 2. マニュアル作成支援

#### Primary Target
**属人化が深刻な中小企業経営者**

```
企業プロフィール:
- 従業員数: 10-50名
- 業界: 製造業、サービス業、小売業
- 年商: 1億-10億円
- 所在地: 地方（県庁所在地以外も含む）
- 経営段階: 創業期を脱し、組織化が必要な段階

経営者プロフィール:
- 年齢: 45-65歳
- 背景: 創業者または2代目経営者
- 課題: 特定の人に依存した業務、技能継承の不安
- 価値観: 堅実経営、地域重視、人間関係重視
```

#### 業種別セグメント
```
1. 町工場・小規模製造業
   - 課題: 職人技の継承、品質の標準化
   - キーパーソン: 熟練工、技術責任者
   - 導入動機: 技術継承の危機感

2. 地域密着サービス業（美容室、整体、飲食等）
   - 課題: 接客品質のばらつき、新人教育負担
   - キーパーソン: 店長、ベテランスタッフ
   - 導入動機: 品質向上、効率化

3. 専門士業（税理士、行政書士、社労士事務所）
   - 課題: 複雑な手続きの属人化
   - キーパーソン: 代表者、ベテラン職員
   - 導入動機: ミス削減、効率化
```

### 3. AIエージェント構築

#### Primary Target
**技術力のある中堅企業IT部門**

```
企業プロフィール:
- 従業員数: 100-500名
- 業界: IT企業、金融業、製造業（大手）
- 年商: 50億-500億円
- 所在地: 東京、大阪、名古屋等の大都市圏
- IT成熟度: 高い、独自システム開発経験あり

担当者プロフィール:
- 役職: CTO、システム部長、開発マネージャー
- 年齢: 30-45歳
- 背景: エンジニア出身、技術的判断ができる
- 課題: 高度なAI活用、既存システムとの統合
```

### 4. GPTs開発

#### Primary Target
**個人事業主・小規模チーム**

```
個人プロフィール:
- 業種: コンサルタント、マーケター、デザイナー、士業
- 年収: 500万-2000万円
- 年齢: 25-45歳
- 働き方: フリーランス、小規模事業者
- ITリテラシー: 中程度、新しいツールに積極的

チームプロフィール:
- 規模: 2-10名
- 業種: デザイン会社、マーケティング会社、コンサル会社
- 特徴: 効率化への関心が高い、ROI重視
```

### 5. IoT & 物理世界AI

#### Primary Target
**一次産業・地方自治体**

```
一次産業:
- 業種: 農業、畜産業、水産業
- 規模: 家族経営-中規模法人
- 課題: 人手不足、効率化、品質向上
- 地域: 全国（特に中山間地域）

地方自治体:
- 規模: 人口1万-10万人の市町村
- 課題: 住民サービス向上、効率化、人口減少対策
- 予算: 限定的、効果の明確な説明が必要
```

## ターゲット別アプローチ戦略

### 1. 企業規模別戦略

#### 小規模企業（10-50名）
```
Priority: マニュアル作成支援、GPTs開発
Approach:
- 感情的訴求（安心感、継承の価値）
- 経営者直接アプローチ
- 地域密着型営業
- 口コミ・紹介重視

Message:
"会社の大切な技術を、確実に次世代に継承しませんか？"
```

#### 中規模企業（50-200名）
```
Priority: AIチーム開発伴走支援
Approach:
- ROI重視の提案
- DX推進担当者経由
- 段階的導入提案
- 成功事例重視

Message:
"社内にAIチームを作り、外部依存から脱却しませんか？"
```

#### 中堅企業（200-500名）
```
Priority: AIエージェント構築
Approach:
- 技術的優位性訴求
- CTO・技術責任者アプローチ
- カスタマイズ性強調
- 競争優位性訴求

Message:
"業界をリードする独自AIシステムを構築しませんか？"
```

### 2. 業界別戦略

#### 製造業
```
共通課題: 技能継承、品質安定、効率化
Entry Point: マニュアル作成 or AIチーム開発
Expansion: エージェント構築（生産管理、品質管理）
```

#### サービス業
```
共通課題: 人材不足、サービス品質、効率化
Entry Point: マニュアル作成 or GPTs開発
Expansion: AIチーム開発（顧客対応自動化）
```

#### IT・テック企業
```
共通課題: 開発効率化、技術的差別化
Entry Point: エージェント構築
Expansion: IoT活用（新規事業）
```

### 3. 地域別戦略

#### 東京都心部
```
特徴: 高いITリテラシー、競争激化
Focus: エージェント構築、GPTs開発
Approach: 技術的優位性、先進性訴求
```

#### 地方都市
```
特徴: 堅実志向、人間関係重視
Focus: AIチーム開発、マニュアル作成
Approach: 信頼関係構築、地域密着
```

#### 中山間地域
```
特徴: 人口減少、一次産業中心
Focus: IoT & 物理世界AI、マニュアル作成
Approach: 地域課題解決、持続可能性
```
