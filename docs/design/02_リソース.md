# 開発者向けデザインリソース集

**Source**: [bradtraversy/design-resources-for-developers](https://github.com/bradtraversy/design-resources-for-developers)

LP制作やUI実装において、素材やツールが必要になった際に参照するためのリストです。

## 主なカテゴリ
*   **[UI Graphics](https://github.com/bradtraversy/design-resources-for-developers#ui-graphics)**: UIキット、グラフィック素材
*   **[Fonts](https://github.com/bradtraversy/design-resources-for-developers#fonts)**: フォントリソース
*   **[Colors](https://github.com/bradtraversy/design-resources-for-developers#colors)**: カラーパレット、グラデーション生成
*   **[Icons](https://github.com/bradtraversy/design-resources-for-developers#icons)**: アイコンライブラリ
*   **[Stock Photos](https://github.com/bradtraversy/design-resources-for-developers#stock-photos)**: ストックフォト
*   **[Illustrations](https://github.com/bradtraversy/design-resources-for-developers#vectors--clip-art)**: イラスト、ベクター素材
*   **[CSS Frameworks](https://github.com/bradtraversy/design-resources-for-developers#css-frameworks)**: CSSフレームワーク
*   **[React UI Libraries](https://github.com/bradtraversy/design-resources-for-developers#react-ui-libraries)**: React用UIライブラリ

## 活用シーン
*   LPの挿絵が欲しい時
*   配色に迷った時
*   アイコンを探している時
