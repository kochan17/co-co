## 共通デザインシステム (Claude Design Skill準拠)

### Core Craft Principles
| 項目 | 仕様 |
|------|------|
| **Grid** | 4pxベース (`4`, `8`, `12`, `16`, `24`, `32px`) |
| **Padding** | 対称であること (TLBR一致) |
| **Border Radius** | 4pxグリッドに従う (`4`, `8`, `12px`) |
| **Animation** | 150-250ms, `cubic-bezier(0.25, 1, 0.5, 1)` |

### Typography
| 用途 | フォント | 特徴 |
|------|----------|------|
| **基本** | `Inter`, `Noto Sans JP` | Geometric Sans, 読みやすさ重視 |
| **データ/技術** | `JetBrains Mono` | 等幅, 数字・コード・キーワード |

### Iconography
- **推奨**: [Phosphor Icons](https://phosphoricons.com/) (`@phosphor-icons/react`)
- 代替: [Lucide](https://lucide.dev/)

## 1. AIチーム開発伴走支援

| 項目 | 内容 |
|------|------|
| **ステータス** | 設計中 |
| **ターゲット** | 中小・中堅企業の経営層、DX推進担当者 |
| **Design Direction** | **Sophistication & Trust** + **Precision & Density** |
| **Color Foundation** | Cool (Slate/Blue-gray), **Dark Mode** |
| **Emotional Job** | 信頼、先進性、自律 |

### コンセプト: "Intellectual Modern"
知的で落ち着きがありつつ、最先端の技術を感じさせる。ありがちな青白のIT系LPではなく、ダークモードを基調とした没入感のある世界観。

### リファレンス
| サイト | 参考ポイント |
|--------|--------------|
| [バイテック生成AI](https://generative-ai.bytech.jp/) | ダークモード、発光表現、3D |
| [テックキャンプ AI活用支援](https://tech-camp.in/ai-solution) | 構成、キーワード、Before/After事例 |

### Color Palette
```
Background:  #0a0a0a (Deep Black), #171717 (Surface)
Accent:      #3b82f6 (Tech Blue) → #8b5cf6 (Purple) グラデーション
Text:        #ededed (Primary), #a1a1aa (Secondary), #525252 (Tertiary)
Border:      rgba(255,255,255,0.1)
```

### コンテンツ構成
1. **Hero**: 「社内にAIチームを作り、外部依存からの脱却を」
   - Visual: Three.js 3Dパーティクルネットワーク
   - Interaction: マウス追従で視点移動
2. **Target**: ビジネス職 / エンジニア職それぞれの課題
3. **Reason**: 選ばれる理由 (専任メンター, 内製化支援, 助成金)
4. **Services**: 伴走支援 / AIエージェント開発 / マニュアル作成
5. **Use Cases**: 具体的な業務効率化事例 (Before/After)
6. **CTA**: 資料請求

### UIパターン
| セクション | パターン | 演出 |
|------------|----------|------|
| Hero | Full-screen Canvas | 3Dパーティクル |
| Cards | Glassmorphism | Hover時Border Glow |
| Motion | Framer Motion | Scroll Fade-in |
| CTA | Gradient Border Button | |
