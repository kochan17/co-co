# リスキリング補助金制度 営業資料向けサマリー

## 要点まとめ

### 結局何ができるのか

**企業が当社の研修サービスを導入する際、研修費用の最大75%を国や自治体が補助します。**

- 例:100万円の研修 → **企業の実質負担25万円**
- さらに研修中の人件費も1時間あたり最大1,000円補助
- 当社が準備すべきことは**研修内容の要件整備とLMS機能の強化**のみ

---

## 私たちは何を準備すればいいのか

当社のサービスを助成金対象として提供するためには、以下の準備が必要となる。まず必須対応として取り組むべき項目を整理した。

### 必須対応

| 項目                          | 内容                                                                                                 | 期限目安  |
| ----------------------------- | ---------------------------------------------------------------------------------------------------- | --------- |
| **1. 研修時間の設計**   | 最低10時間以上のカリキュラム構成                                                                     | 即時      |
| **2. LMS機能の実装**    | • 受講ログの秒単位記録\u003cbr\u003e• 本人確認機能(顔認証推奨)\u003cbr\u003e• CSV形式でのログ出力 | 3ヶ月以内 |
| **3. 研修証明書の発行** | 修了証、受講証明書のテンプレート作成                                                                 | 1ヶ月以内 |

必須対応に加えて、以下の推奨対応を行うことで、営業時の訴求力を高めることができる。

### 推奨対応

| 項目                             | 内容                               | メリット                 |
| -------------------------------- | ---------------------------------- | ------------------------ |
| **経産省Reスキル講座認定** | 「国が認めた講座」ブランド獲得     | 営業時の信頼性向上       |
| **マナビDX登録**           | IPA運営のポータルサイトへの掲載    | 認知度向上・認定への近道 |
| **企業向けサポート資料**   | 助成金申請ガイド、テンプレート提供 | 導入ハードル低減         |

## 営業資料には何を書けばいいのか

営業資料では、助成金活用による具体的なメリットを数字で示すことが重要である。以下のポイントを押さえて訴求する。

### セールスポイント

営業資料の見出しでは、以下のようなコピーが効果的である。

#### 見出し例

```
・研修費用の最大75%を国が補助
・100万円の研修が実質25万円で導入可能
・研修中の人件費も1時間1,000円補助
```

抽象的な説明ではなく、具体的な金額シミュレーションを提示することで、導入の意思決定を後押しできる。

#### 具体的な数字で訴求

**ケース1:中小企業が10名に研修を実施する場合**

| 項目                            | 金額            |
| ------------------------------- | --------------- |
| 研修費用(定価)                  | 100万円         |
| 経費補助(75%)                   | ▲75万円        |
| 賃金補助(20時間×1,000円×10名) | ▲20万円        |
| **実質企業負担**          | **5万円** |

**ケース2:東京都の中小企業の場合**

| 項目                            | 金額               |
| ------------------------------- | ------------------ |
| 研修費用                        | 50万円             |
| 東京都DXリスキリング助成金(75%) | ▲37.5万円         |
| **実質企業負担**          | **12.5万円** |

企業の状況によって最適な制度は異なる。以下のフローチャートで判断基準を示す。

### どの制度を選べばいいのか

```
Q1: 企業の所在地は?
├─ 東京都内 → 「東京都DXリスキリング助成金」を第一候補に
└─ その他 → 「人材開発支援助成金」を第一候補に

Q2: 研修の目的は?
├─ 新規事業・DX推進 → 「事業展開等リスキリング支援コース」(補助率75%)
├─ 一般的なスキルアップ → 「人材育成支援コース」(補助率45%)
└─ 転職を伴う個人向け → 「経産省リスキリング支援事業」(補助率70%)
```

## 主要制度の比較表

主要な3つの制度について、重要なポイントを一覧で比較した。営業資料にそのまま活用できる。

| 制度名                         | 管轄   | 補助率  | 上限額      | 対象企業         | 特徴                     |
| ------------------------------ | ------ | ------- | ----------- | ---------------- | ------------------------ |
| **人材開発支援助成金**   | 厚労省 | 最大75% | 年1,000万円 | 全国の企業       | 賃金補助あり(1h/1,000円) |
| **東京都DXリスキリング** | 東京都 | 75%     | 年100万円   | 都内中小企業     | 申請が比較的簡単         |
| **経産省リスキリング**   | 経産省 | 最大70% | 56万円/人   | 個人(転職希望者) | 転職成功で追加補助       |

## 企業はどのように申請すればいいのか

助成金申請の流れは制度によって異なるが、ここでは最も利用頻度の高い人材開発支援助成金の申請フローを示す。

### 人材開発支援助成金の場合

```
【研修開始1ヶ月前まで】
① 訓練計画届を労働局に提出
   ↓
【研修実施】
② 当社の研修を受講(出席記録・ログ管理)
   ↓
【研修終了後2ヶ月以内】
③ 支給申請書を労働局に提出
   ↓
【2〜3ヶ月後】
④ 助成金が企業に振り込まれる
```

企業の申請負担を軽減するため、当社では以下のサポートを提供する。

### 当社がサポートできること

- 訓練カリキュラム(シラバス)の提供
- 受講証明書・修了証の発行
- 受講ログデータのCSV出力
- 申請用テンプレートの提供
- 提携社労士の紹介(オプション)

---

## 何に注意すればいいのか

助成金申請では、些細なミスが不支給につながることがある。よくある失敗パターンとその対策を整理した。

### 企業側の失敗パターン

| 失敗例           | 対策                                   |
| ---------------- | -------------------------------------- |
| 計画届の提出忘れ | 研修開始1ヶ月前の期限厳守をリマインド  |
| 出席記録の不備   | 当社LMSで自動記録・証明書発行          |
| 研修中の業務兼務 | 「研修中は業務から完全に離れる」を徹底 |

当社側でも、以下の点に注意して研修を設計する必要がある。

### 当社側の失敗パターン

| 失敗例               | 対策                       |
| -------------------- | -------------------------- |
| 研修時間が10時間未満 | 最低10時間以上のコース設計 |
| 受講ログが不十分     | 秒単位のログ記録機能を実装 |
| 本人確認ができない   | 顔認証ログイン機能の導入   |

## なぜ他社は成功しているのか

競合他社の成功事例から学ぶべきポイントを分析した。特に助成金活用を営業戦略の中核に据えている企業の取り組みが参考になる。

### Aidemy Business

- **戦略**: サイトに「助成金対応」を大きく表示
- **実績**: 「研修費の70%補助」を前面に出して導入促進
- **学び**: 価格ハードルを下げる最強の武器

### AVILEN

- **戦略**: 経産省Reスキル講座認定を取得
- **実績**: 「国が認めた講座」として大手800社に導入
- **学び**: 公的認定がブランド価値を高める

### Schoo for Business

- **戦略**: 助成金申請用の書類を管理画面から自動生成
- **実績**: 申請の手間を削減し、B2B営業の差別化に成功
- **学び**: 企業の申請負担を減らすサポート機能が重要

## どのように営業すればいいのか

助成金活用を前面に出した営業トークの具体例を示す。初回提案時と、よくある質問への回答パターンを用意した。

### 初回提案時

```
「当社の研修は、国の助成金制度を活用いただくことで、
研修費用の最大75%が補助されます。

例えば100万円の研修であれば、御社の実質負担は25万円。
さらに研修中の人件費も1時間あたり1,000円補助されるため、
20時間の研修を10名で実施した場合、20万円の賃金補助が受けられます。

つまり、100万円の研修が実質5万円で導入可能です。

申請手続きは当社が全面サポートいたしますので、
御社の負担は最小限に抑えられます。」
```

企業からよく寄せられる質問とその回答例を以下にまとめた。

### よくある質問への回答

**Q: 申請は難しくないですか?**

```
A: 当社が申請に必要な書類(カリキュラム、受講証明書、ログデータ)を
すべて提供いたします。また、提携の社労士もご紹介可能ですので、
初めての企業様でも安心してご利用いただけます。
```

**Q: 助成金が出るまでどれくらいかかりますか?**

```
A: 研修終了後、申請から2〜3ヶ月で御社に振り込まれます。
研修費用は一旦全額お支払いいただく必要がありますが、
分割払いなどのご相談も承ります。
```

## 次に何をすればいいのか

助成金対応を実現するための具体的なアクションプランを、短期・中期・長期に分けて整理した。

### 短期
1. 研修カリキュラムを10時間以上に再設計
2. 修了証・受講証明書のテンプレート作成
3. 営業資料に助成金活用事例を追加

### 中期
1. LMSに受講ログ記録機能を実装
2. 顔認証ログイン機能の導入検討
3. 企業向け申請サポートガイドの作成

### 長期
1. 経産省「マナビDX」への登録申請
2. Reスキル講座認定の取得
3. 提携社労士ネットワークの構築
